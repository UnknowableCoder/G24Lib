Detailed Description

template<class T, class indexer = g24_lib::Definitions::default_unsigned_indexer>
struct g24_lib::Parallelism::parallelism_base::reduce_temporaries< T, indexer >

The class that is suitable to hold temporary values during reductions.

Template Parameters
TThe type of the values being reduced.
indexerA type that is appropriate to hold the size of the container.
Precondition
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
