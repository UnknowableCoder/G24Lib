Detailed Description

template<class indexer, class factorT, g24_lib::Definitions::default_unsigned_indexer ord, g24_lib::Definitions::default_unsigned_indexer npoints>
class g24_lib::derivator< indexer, factorT, ord, npoints >

A 1D differential operation of order ord weighted by npoints.

Template Parameters
indexerA type that is appropriate to hold the indexes of the points.
factorTThe factor to be used in the coefficients.
Precondition
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
factor_type must support multiplications and divisions with itself (and with the elements of the array whose derivatives will be evaluated). It does not need to be a floating point type since the coefficients are stored essentially as rational numbers.
Warning
Currently, the library only gives correct results for values of order up to (and including) 6.
Currently, the library only gives correct results for values of npoints up to (and including) order + 7.
Remarks
The accuracy of the derivative will be given by npoints - order.

Parameters
iThe index of the point under consideration.
offsetThe offset from the leftmost point at which the derivative is being evaluated.
Remarks
For instance, if the derivative would be calculated with 7 points and be evaluated at the central point, offset would be 3.
Parameters
dividedIf true, returns the coefficient proper; if false, returns only the numerator, with the denominator being obtainable through coefficient_denominator.
