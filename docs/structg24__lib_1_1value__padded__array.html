Detailed Description

template<class Cont, bool by_ref = true, class Val = g24_lib::value_type<Cont>, class indexer = g24_lib::Definitions::default_unsigned_indexer>
struct g24_lib::value_padded_array< Cont, by_ref, Val, indexer >

A container padded with a fixed value after a specified size.

Template Parameters
ContThe container to be padded.
by_refSpecifies if the original container is stored by value or by reference. Most sane use cases should work with references (provided they remain valid throughout the value_padded_array's lifetime), but that is not the case when one is passing parameters into CUDA GPU kernels, since references become invalid.
ValThe type of the padding value.
Precondition
Cont::operator [] must yield something that can be casted to or from a Val.
indexer must be a valid choice for g24_lib::Definitions::default_unsigned_indexer.
Remarks
This comes in rather handy when one wants to operate on arrays of different sizes and assign a neutral value to elements after the end of the smallest. For instance, when summing two arrays of different sizes, any values after the end of the smallest can be put to zero.
