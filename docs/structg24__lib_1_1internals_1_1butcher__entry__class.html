Detailed Description

template<class factorT, class eval_gen, class... frac_gens>
struct g24_lib::internals::butcher_entry_class< factorT, eval_gen, frac_gens >

An entry of a Butcher tableau.

Template Parameters
factorTThe type that will hold the parameters (numerator and denominator).
eval_genA type whose value() static method yields a g24_lib::fraction<factorT> which corresponds to the evaluation time.
frac_gensA set of types whose value() static method yields a g24_lib::fraction<factorT> which correspond to the coefficients of the previous evaluations of the derivative function.
Remarks
A Butcher tableau entry of the form a | b c d ... would be represented by butcher_entry_class<T, x<a>, x<b>, x<c>, x<d> ...>, where x<y> is a class whose x::value() is constexpr and returns y. In practice, x should correspond to a g24_lib::constexpr_fraction<factorT, num, dem> for y = num/dem.
