Detailed Description

No parallelism at all: simple, serial C++ code.

Remarks
Since there is no parallelism, the functions in g24_lib::Parallelism::None::atomic are simple operations. They are not thread safe, that is, using g24_lib::Parallelism::None on the same variables in two different threads will lead to memory races!
