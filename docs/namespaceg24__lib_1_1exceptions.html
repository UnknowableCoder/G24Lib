inline

Exception thrown when an error is encountered during input.

Parameters
eThe exception to be thrown.

If g24_lib::Definitions::throw_exceptions is true and the code isn't being ran on a CUDA kernel, throws the exception.

If g24_lib::Definitions::print_exceptions is true or g24_lib::Definitions::throw_exceptions is false, and the code isn't being ran on a CUDA kernel, prints the error message associated with the expression and exits the program with EXIT_FAILURE.

If the code is being ran on a CUDA kernel, regardless of the values of g24_lib::Definitions::throw_exceptions and g24_lib::Definitions::print_exceptions, prints the error message associated with the expression and calls __trap(), which interrupts any running threads, stopping the kernel.
