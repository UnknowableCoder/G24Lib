Detailed Description

template<class T>
struct g24_lib::nested_imitation_struct< T >

Constructs a struct that has the members of a class based on compile time reflection from g24_lib::class_info.

Template Parameters
TThe class being imitated.

If T satisfies the conditions for being a scalar (if it is a bool, integer, floating point type, enum or pointer), then it just holds a variable of the appropriate type. Else, each member is held as a nested_imitation_struct itself, so it can handle structs of structs.

The types are in the same order as g24_lib::class_info<T>::type.

Remarks
Take
struct example_inner
{
int a;
double b;
char *c;
};
struct example_outer
{
float first;
example_inner second, third;
char fourth;
};
We essentially have something equivalent to:
struct g24_lib::nested_imitation_struct<example_outer>
{
g24_lib::nested_imitation_struct<float> one;
g24_lib::nested_imitation_struct<example_inner> two, three;
g24_lib::nested_imitation_struct<char> four;
};
struct g24_lib::nested_imitation_struct<example_inner>
{
g24_lib::nested_imitation_struct<int> one;
g24_lib::nested_imitation_struct<double> two;
g24_lib::nested_imitation_struct<char *> three;
};
struct g24_lib::nested_imitation_struct<float>
{
float one;
};
//And similarly for other native (scalar) types.
Through access to these members is through the get_element() or get() member functions.
