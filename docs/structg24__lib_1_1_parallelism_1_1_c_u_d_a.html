Detailed Description

template<class estimator = CUDA_kernel_size::estimator::automatic, g24_lib::Definitions::default_unsigned_indexer BlockSize = 128>
struct g24_lib::Parallelism::CUDA< estimator, BlockSize >

Parallelised operations using CUDA.

Warning
g24_lib::parallelism::CUDA::atomic uses the CUDA API to provide most operations, so it might fail for user-defined types. For most use cases, we should be dealing with numerical code that deals with native types, so this shouldn't be too problematic.
