Detailed Description

template<class Type>
class g24_lib::comp< Type >

A complex number.

Template Parameters
TypeThe underlying type.
Precondition
Type should have usual arithmetic operators.

This class holds the complex numbers in two simultaneous representations:

Cartesian form: a + b*i

Polar form: r * exp(i*theta) (theta in fractions of pi, that is, in the range ]-1, 1])

Though this doubles memory requirements, it allows for easier computations, using the cartesian form for additions and subtractions and the polar form for multiplications and divisions.

Remarks
Since the standard library already implements std::complex, this might be more than a bit superfluous unless CUDA interoperatability is significant...
