Detailed Description

template<class T>
struct g24_lib::flat_imitation_struct< T >

Constructs a struct that has every scalar member of all the members of a class (at all levels of inclusion) based on compile time reflection from g24_lib::class_info.

Template Parameters
TThe class being imitated.

Every scalar member of any of the members of the class is held by the imitation struct, with every member of T being (recursively) reflected to extract all the scalar types.

The types are in the same order as g24_lib::class_info<T>::true_type.

Remarks
Take
struct example_inner
{
int a;
double b;
char *c;
};
struct example_outer
{
float first;
example_inner second, third;
char fourth;
};
We essentially have something equivalent to:
struct g24_lib::flat_imitation_struct<example_outer>
{
float one;
int two;
double three;
char *four;
int five;
double six;
char *seven;
char eight;
};
struct g24_lib::flat_imitation_struct<example_inner>
{
int one;
double two;
char *three;
};
struct g24_lib::flat_imitation_struct<float>
{
float one;
};
//And so on.
Through access to these members is through the get_element() or get() member functions.
