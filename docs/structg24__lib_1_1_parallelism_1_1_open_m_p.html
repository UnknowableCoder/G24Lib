Detailed Description

Parallelised operations using OpenMP.

Warning
g24_lib::Parallelism::OpenMP::atomic uses #pragma omp atomic, so it might possibly fail for non-scalar type, though the possible loss in performance by using #pragma omp critical instead far outweighs the gains in generality...
