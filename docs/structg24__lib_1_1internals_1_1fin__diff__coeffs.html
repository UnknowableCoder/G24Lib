Detailed Description

template<class indexer, class factor_type, g24_lib::Definitions::default_unsigned_indexer order, g24_lib::Definitions::default_unsigned_indexer npoints>
struct g24_lib::internals::fin_diff_coeffs< indexer, factor_type, order, npoints >

Defines the finite difference coefficients for a given derivative.

Template Parameters
indexerA type that is appropriate to hold the offsets.
factor_typeA type that is appropriate to hold the coefficients.
orderThe order of the derivative.
npointsThe number of points that will be used to calculate the derivative
Precondition
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
factor_type must support multiplications and divisions with itself (and with the elements of the array whose derivatives will be evaluated). It does not need to be a floating point type since the coefficients are stored essentially as rational numbers.
Warning
Currently, the library only gives correct results for values of order up to (and including) 6.
, the library only gives correct results for values of npoints up to (and including) order + 7.
Remarks
The accuracy of the derivative will be given by npoints - order.
