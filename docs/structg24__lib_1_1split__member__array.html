Detailed Description

template<class Type, class indexer = g24_lib::Definitions::default_unsigned_indexer, template< class, class > class memory_manager_type = default_memory_manager, bool default_is_copy = true>
struct g24_lib::split_member_array< Type, indexer, memory_manager_type, default_is_copy >

A container that stores the different members of a struct that can undergo compile time reflection through g24_lib::class_info as separate arrays. In short, translates an array of structs into a struct of arrays in a more or less automatic way.

Template Parameters
TypeThe type of the structs that will be stored.
indexerA type that is appropriate to hold the size of the arrays.
memory_manager_typeThe memory manager corresponding to the memory context where the arrays will be held.
default_is_copySee g24_lib::simple_array.
Precondition
Type must support the limited form of compile time reflection provided by g24_lib::class_info.
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
memory_manager_type must be a valid template for memory managers, that is, g24_lib::is_memory_manager<memory_manager_type<T, indexer>> should be true for any T.
