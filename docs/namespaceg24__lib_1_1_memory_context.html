Detailed Description

The different memory contexts supported by the library are here.

Remarks
Since this was written with just CPU and CUDA code in mind, it is assumed that it can be determined at compile time where the memory will be allocated. More complex memory architectures might eventually need a more flexible approach, but, for the time being, no revision is planned.
