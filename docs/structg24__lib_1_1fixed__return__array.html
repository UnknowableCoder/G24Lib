Detailed Description

template<class Type, class indexer = g24_lib::Definitions::default_unsigned_indexer>
struct g24_lib::fixed_return_array< Type, indexer >

A mock array with a fixed return value.

Precondition
indexer must be a valid choice for g24_lib::Definitions::default_unsigned_indexer.
Remarks
This comes in rather handy when one wants to operate on arrays with a constant, known value (such as 0) without actually allocating memory for them.
