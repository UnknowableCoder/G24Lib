Detailed Description

template<class Type, class indexer>
struct g24_lib::memory_manager_base< Type, indexer >

Provides a base class for all memory managers.

Template Parameters
TypeThe type of the object being managed.
indexerA type that is appropriate to hold sizes of objects.
Precondition
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
Remarks
All of the member functions (and the member typedef) should be overriden in any derived class.
Warning
By design, memory managers should only have global state, if any state at all, since all the functions are static (and called as such in the code: no separate instances of a manager are kept for each container, contrary to what the standard mandates should happen with the STL containers). In all the use cases considered, this was not problematic at all, but one can never accurately predict all future needs...
