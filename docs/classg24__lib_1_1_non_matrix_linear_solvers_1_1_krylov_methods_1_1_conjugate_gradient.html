Detailed Description

template<class parallelism_type = Parallelism::None>
class g24_lib::NonMatrixLinearSolvers::KrylovMethods::ConjugateGradient< parallelism_type >

The Conjugate Gradient Method is reasonably easy to implement, but more efficient than the usual Gauss-Seidel or Jacobi iterative methods. However, it requires symmetrical systems/matrices.

Template Parameters
parallelism_typeA valid type of parallelism, that is, one that satisfies g24_lib::is_parallelism, with which to perform the computations.
