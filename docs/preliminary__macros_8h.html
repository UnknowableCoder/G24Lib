◆ G24_LIB_CLEAN_UP_MACROS

#define G24_LIB_CLEAN_UP_MACROS   0

Controls if all the macros defined in the library will be #undef ined after use. Default is false.

Remarks
The default value is false since some of these macros might be useful and all of them are prefixed by G24_LIB_.
But we still give this option in case there is a strange conflict happening or the user somehow is an anti-macro fundamentalist.
