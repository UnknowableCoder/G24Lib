Parameters

argsExtra arguments to be passed to op.
Template Parameters
ContThe type of the container through which we will loop.
OpThe type that holds the operation to be applied to the elements of the container.
ArgsThe types for any extra arguments to Op.
Precondition
An object of type Op must take as arguments (a reference to) c, the index at which we're evaluating and any extra arguments passed to this function in args.
Remarks
Since, in principle, the only run-time component that will influence the kernel size will be the size of the container, this is the only argument that is actually used. The others serve mainly to enable argument type deduction when the user already has every argument instantiated and simply does not want to write the types explicitly again. Any sane optimizing compiler should optimize them out...
