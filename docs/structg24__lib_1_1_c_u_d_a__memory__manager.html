Detailed Description

template<class Type, class indexer = g24_lib::Definitions::default_unsigned_indexer>
struct g24_lib::CUDA_memory_manager< Type, indexer >

Manages memory on a CUDA-enabled GPU.

Template Parameters
TypeThe type of the object being managed.
indexerA type that is appropriate to hold sizes of objects.
Precondition
indexer must satisfy the constraints placed on the choice of g24_lib::Definitions::default_unsigned_indexer.
