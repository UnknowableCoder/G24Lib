Detailed Description

template<class Op, class indexer = g24_lib::Definitions::default_unsigned_indexer>
struct g24_lib::operation_array< Op, indexer >

A mock array with its return value being calculated at each point.

Template Parameters
OpFunction, functor, lambda or any other object that will yield the desired return value when operated with an indexer: op(i)
Precondition
indexer must be a valid choice for g24_lib::Definitions::default_unsigned_indexer.
Remarks
This comes in rather handy when one wants to operate on arrays whose elements can be easily calculated without actually allocating memory for them.
For most practical use cases, functors or lambdas will be preferred for Op, since some form of state or external parameters might be required to properly calculate the elements at each index.
